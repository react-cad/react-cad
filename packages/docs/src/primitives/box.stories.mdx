# box

The `box` element creates an axis-aligned cuboid at the origin. Width, length and height are controlled by the `x`, `y` and `z` props. By [default](#default-box) the box has one corner at the origin and extends in the positive x-, y- and z-directions, but it can optionally be [centered](#centered-box) on the origin with the `center` prop.
