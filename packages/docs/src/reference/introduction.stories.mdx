# Reference

ReactCAD uses a left-handed co-ordinate system. `z` is up, `x` is right and `y` points out of the screen. This slightly unusual arrangement allows easy creation of 2d shapes from SVGs.

## Elements

### <Link to="reference-operations-introduction">Operations</Link>

Operations combine multiple 3D solids into one solid, by adding, subtracting or taking common volumes.

- <Link to="reference-operations-union">Union</Link>
- <Link to="reference-operations-difference">Difference</Link>
- <Link to="reference-operations-intersection">Intersection</Link>

### <Link to="reference-primitives-introduction">Primitives</Link>

Primitives are basic solids which can be combined to build CAD models.

- <Link to="reference-primitives-box">Box</Link>
- <Link to="reference-primitives-sphere">Sphere</Link>
- <Link to="reference-primitives-cylinder">Cylinder</Link>
- <Link to="reference-primitives-cone">Cone</Link>
- <Link to="reference-primitives-torus">Torus</Link>
- <Link to="reference-primitives-wedge">Wedge</Link>
- <Link to="reference-primitives-polyhedron">Polyhedron</Link>
- <Link to="reference-primitives-solid">Solid</Link>

### <Link to="reference-sweeps-introduction">Sweeps</Link>

Sweeps create more complex solids by sweeping a 2D shape along a path to form a solid volume.

- <Link to="reference-sweeps-prism">Prism</Link>
- <Link to="reference-sweeps-helix">Helix</Link>
- <Link to="reference-sweeps-revolution">Revolution</Link>
- <Link to="reference-sweeps-pipe">Pipe</Link>
- <Link to="reference-sweeps-evolution">Evolution</Link>
- <Link to="reference-sweeps-loft">Loft</Link>

### <Link to="reference-imports-introduction">Imports</Link>

Imports create 3d solids by importing various CAD file formats.

- <Link to="reference-imports-brep">BREP</Link>
- <Link to="reference-imports-step">STEP</Link>
- <Link to="reference-imports-stl">STL</Link>
- <Link to="reference-imports-obj">OBJ</Link>

### <Link to="reference-transformations-introduction">Transformations</Link>

Transformations move, rotate, scale, shear or mirror other shapes.

- <Link to="reference-transformations-translation">Translation</Link>

- <Link to="reference-transformations-scale">Scale</Link>
- <Link to="reference-transformations-mirror">Mirror</Link>
