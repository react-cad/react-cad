# revolution

The `revolution` element extrudes a 2-dimensional shape by rotating it around an axis. It is rotated by `angle` radians around an `axis` defined by a 3d vector. The 2d shape is passed to the `profile` prop, which can either be a list of points describing a planar polygon, or a string containing an SVG image. The `profile` must not intersect the `axis`.

## Examples

export const Revolution = (props) => <revolution {...props} />;

### Triangular revolution

A triangular profile rotated in a full circle around the y-axis.

<Source language="jsx" code={`
  <revolution axis={[0,1,0]} angle={2 * Math.PI} profile={[[1,0,0], [3,0,0], [2,2,0]]} />
`} />

<Story name="revolution" args={{ axis: [0,1,0], angle: 2 * Math.PI, profile: [[1,0,0], [3,0,0], [2,-2,0]] }}>
  {Revolution.bind({})}
</Story>

### Partial elliptical revolution

An elliptical profile rotated 3 quarters of a turn around a diagonal axis.

<Source language="jsx" code={`
  <revolution axis={[1,0,1]} angle={3 * Math.PI / 2} profile={\`<svg><ellipse rx="2" ry="3" /></svg>\`} />
`} />

<Story name="elliptical revolution" args={{ axis: [1,0,1], angle: 3 * Math.PI / 2, profile: `<svg><ellipse rx="2" ry="3" /></svg>` }}>
  {Revolution.bind({})}
</Story>

### React icon revolution

An complex shape rotated 1 eighth turn around the x-axis.
