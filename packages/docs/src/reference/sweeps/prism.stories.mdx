# prism

The `prism` element extrudes a 2-dimensional shape along a vector. The vector is defined either by the `x`, `y` and `z` props or by the `vector` prop. The children of the `prism` form the 2d shape (see <Link to="reference-surfaces-svgs--introduction">Surfaces & SVGs</Link>).

## Examples

export const Prism = (props) => <prism {...props} />;

### Triangular prism

A prism of height 5 with a triangular profile.

<Source
  language="jsx"
  code={`
<prism z={5}>
  <planar>
    <svg>
      <polygon points="0,0 3,0 1.5,2" />
    </svg>
  </planar>
</prism>
`}
/>
